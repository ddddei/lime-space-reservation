# FAQ

참가자와 운영자가 자주 묻는 질문을 모았습니다. 문제가 계속되면 [TROUBLESHOOTING.md](./TROUBLESHOOTING.md)의 확인 순서를 따라가 주세요.

## 참가자 FAQ

### 예약이 안 됩니다.

아래 순서대로 확인해 주세요.

1. 관리자 승인 완료 상태인지 확인합니다.
2. 선택한 시간이 공간 운영시간 안인지 확인합니다. 24시간 운영 공간은 실제 운영시간과 별개로 예약 신청은 09:00~22:00까지만 가능합니다.
3. 운영자가 막아 둔 차단 일정과 겹치지 않는지 확인합니다.
4. 이미 예약된 시간인지 확인합니다.
5. 최대 이용시간을 넘지 않았는지 확인합니다.
6. 계속 안 되면 Discord 문의 채널 또는 운영자 DM으로 문의합니다.

### 시간은 어떤 단위로 선택하나요?

30분 단위로 선택합니다. 원하는 시작 시각 버튼을 눌러 연속된 구간을 만들면 됩니다. 하루 최대 4시간(30분 슬롯 8개)까지 선택할 수 있습니다.

### 로그인이 안 됩니다.

1. 이름을 등록된 이름과 똑같이 입력했는지 확인합니다.
2. 전화번호 전체를 정확히 입력했는지 확인합니다.
3. 등록된 참가자인지 운영자에게 확인합니다.
4. 계속 안 되면 운영자에게 이름과 전화번호 확인을 요청합니다.

### 사진이 안 보입니다.

1. 페이지를 새로고침합니다.
2. 다시 로그인합니다.
3. 다른 네트워크에서 접속해 봅니다.
4. 계속 안 보이면 운영자에게 문의합니다.

### 내 신청이 안 보입니다.

1. `내 신청 확인/수정`에서 `새로고침`을 누릅니다.
2. 취소한 신청이라면 `취소된 신청 보기`를 켭니다.
3. 신청 완료 메시지를 봤는지 확인합니다.
4. 계속 안 보이면 운영자에게 문의합니다.

### 신청을 취소하고 싶습니다.

1. `내 신청 확인/수정`에서 신청을 찾습니다.
2. `신청 취소`를 누릅니다.
3. 확인 창에서 다시 `신청 취소`를 누릅니다.
4. 취소 후에는 기본 목록에서 숨겨지고, `취소된 신청 보기`에서 확인할 수 있습니다.

## 운영자 FAQ

### 신청 취소는 어떻게 하나요?

1. 관리자 화면의 `전체 신청 목록`에서 신청을 찾습니다.
2. 해당 행의 `취소`를 누릅니다.
3. 확인 모달에서 `신청 취소`를 누릅니다.
4. 참가자에게 `내 신청 확인/수정`에서 `새로고침`을 안내합니다.

### 차단 일정은 언제 쓰나요?

외부 대관, 내부 행사, 점검, 촬영 등으로 참가자가 예약하면 안 되는 시간이 있을 때 사용합니다.

확정되지 않은 일정은 차단하지 말고 관리자 메모로 남긴 뒤, 확정되면 차단 일정으로 저장하세요.

### 공간을 숨기고 싶습니다.

`공간 정보 관리`에서 해당 공간의 `사용자 숨김`을 누릅니다.

사용자 숨김은 참가자 화면에서만 숨깁니다. 데이터가 삭제되는 것은 아니며, 다시 `사용자 노출`로 바꿀 수 있습니다.

### 공간을 운영에서 제외하고 싶습니다.

운영에서 제외할 공간은 `비활성 처리`를 사용합니다.

단순히 잠시 숨기는 목적이면 `사용자 숨김`을 사용하세요.

### 운영 종료 후 무엇을 확인해야 하나요?

1. 신청 목록을 확인합니다.
2. 보존해야 할 운영 기록이 있는지 확인합니다.
3. 테스트 신청 정리가 필요한 경우 개발 담당자에게 요청합니다.
4. `meetings = 0`, `sessions = 0` 상태가 필요한 운영 시작 전인지 확인합니다.
5. 수동 정리 파일은 [supabase/manual-sql](./supabase/manual-sql)에 있습니다.
