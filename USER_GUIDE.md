# 라임 공간 예약 안내

## 로그인

1. 이름을 입력합니다.
2. 전체 전화번호를 입력합니다.
3. `참여자 확인`을 누릅니다.

등록된 참여자만 예약할 수 있습니다.

## 예약하기

1. 원하는 공간 카드를 누릅니다.
2. 예약 모달에서 사진, 공간 정보, 날짜를 확인합니다.
3. 시간을 선택합니다.
4. 모임명을 입력합니다.
5. `모임공간 신청하기`를 누릅니다.

신청 후 `내 신청 확인/수정`에 바로 표시됩니다.

## 예약 취소

1. `내 신청 확인/수정`에서 신청을 찾습니다.
2. `신청 취소`를 누릅니다.
3. 확인 창에서 다시 `신청 취소`를 누릅니다.

취소된 신청은 기본 목록에서 숨겨집니다. 필요하면 `취소된 신청 보기`를 켜서 확인할 수 있습니다.

## 사진 보기

공간 카드 또는 예약 모달의 `사진 보기` 버튼을 누르면 큰 사진 팝업이 열립니다.

- 닫기 버튼
- 바깥 클릭
- ESC

위 방법으로 닫을 수 있습니다.
