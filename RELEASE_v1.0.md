# Release v1.0

## 범위

v1.0은 라임 제휴공간 예약 운영을 위한 첫 운영 버전입니다.

## 포함 기능

- 참여자 로그인
- 관리자 로그인
- 제휴공간 목록 조회
- 공간 카드 기반 예약 진입
- 예약 신청 모달
- 공간 사진 팝업
- 날짜/시간 선택
- 모임명 입력
- 내 신청 확인
- 참가자 신청 취소
- 취소된 신청 보기
- 관리자 전체 신청 조회
- 관리자 신청 취소
- 참여자 예약 승인 상태 확인
- Discord 문의 카드

## 운영 데이터

- 피스메이커 참가자 26명 반영
- Level 2 14명, Level 1 12명 반영
- 관리자 10명 반영
- 운영 오픈 전 `meetings/sessions` 초기화 SQL 정리

## 검증

- 전체 참가자 26명 운영 시나리오 QA
- 관리자 신청 조회/취소 QA
- 운영 DB count 검증
- `npm run build`
- `npm run lint`
- `git diff --check`

## v1.1 이후 변경 참고

현재 main에는 v1.1 운영 기능도 포함되어 있습니다.

- 관리자 새로고침 후 세션 유지
- 관리자 차단 일정 저장/수정/해제
- 공간 사용자 노출/숨김
- 공간 설명, 관리자 메모, 정원, 특징 태그 수정
