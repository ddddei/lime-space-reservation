# 오픈 당일 체크리스트

오픈 전에 위에서 아래로 확인하세요.

- [ ] 관리자 로그인
- [ ] 참가자 로그인
- [ ] 신청 가능
- [ ] 차단 일정 확인
- [ ] 공간 노출 확인
- [ ] Discord 문의 확인
- [ ] `meetings = 0`
- [ ] `sessions = 0`
- [ ] 운영 시작

## 확인 메모

- 관리자 로그인은 우측 상단 `관리자 모드`에서 확인합니다.
- 참가자 로그인은 실제 참가자 계정 1명으로 확인합니다.
- 신청 가능 여부는 테스트 신청을 만들어 확인합니다.
- 테스트 신청을 만들었다면 운영 시작 전 정리 여부를 확인합니다.
- 차단 일정은 예약 불가 시간이 참가자 화면에 반영되는지 확인합니다.
- 공간 노출은 참가자 화면에서 보여야 할 공간만 보이는지 확인합니다.
- Discord 문의 채널은 운영자가 즉시 응답할 수 있는 상태인지 확인합니다.
