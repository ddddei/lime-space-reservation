# 새 운영자 5분 매뉴얼

처음 운영을 맡은 사람이 5분 안에 전체 흐름을 확인하기 위한 문서입니다. 자세한 설명은 [ADMIN_GUIDE.md](./ADMIN_GUIDE.md)를 참고하세요.

## 1. 관리자 로그인

1. 예약 페이지를 엽니다.
2. 우측 상단 `관리자 모드`를 누릅니다.
3. 관리자 이름과 전화번호 전체를 입력합니다.
4. 관리자 화면이 열리는지 확인합니다.

## 2. 전체 신청 목록 확인

1. `전체 신청 목록`을 봅니다.
2. `새로고침`을 눌러 최신 상태로 맞춥니다.
3. 신청자, 모임명, 공간, 날짜, 상태를 확인합니다.
4. 취소된 신청은 상태 필터에서 `취소됨`을 선택해 확인합니다.

## 3. 차단 일정 확인

1. `차단 일정` 영역을 확인합니다.
2. 운영자가 막아야 하는 날짜와 시간이 등록되어 있는지 봅니다.
3. 필요한 경우 공간, 날짜, 시간, 사유를 입력하고 `차단 저장`을 누릅니다.
4. 잘못 등록된 일정은 `수정` 또는 `해제`로 정리합니다.

## 4. 공간 정보 관리 확인

1. `공간 정보 관리`를 엽니다.
2. 참가자에게 보여야 하는 공간이 `사용자 노출` 상태인지 확인합니다.
3. 숨겨야 하는 공간은 `사용자 숨김`으로 바꿉니다.
4. 운영에서 제외할 공간만 `비활성 처리`합니다.

## 5. 운영 시작

1. 참가자 로그인이 되는지 확인합니다.
2. 예약 신청이 가능한지 확인합니다.
3. Discord 문의 채널을 확인할 수 있는 상태로 둡니다.
4. 운영 중 `전체 신청 목록`을 주기적으로 `새로고침`합니다.

## 6. 운영 종료 후 DB 확인

운영 종료 후 또는 다음 운영 시작 전에 신청 데이터 정리가 필요한지 확인합니다.

- `meetings = 0`
- `sessions = 0`

정리가 필요하면 운영자가 직접 처리하지 말고 개발 담당자에게 [supabase/manual-sql](./supabase/manual-sql) 실행을 요청하세요.
