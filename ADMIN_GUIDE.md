# 관리자 운영 가이드

## 관리자 로그인

1. 우측 상단 `관리자 모드`를 누릅니다.
2. 관리자 이름과 전체 전화번호를 입력합니다.
3. 로그인 후 관리자 화면으로 이동합니다.

관리자 로그인은 현재 브라우저 탭의 `sessionStorage`에만 유지됩니다. 운영 중 새로고침해도 관리자 화면, 전체 신청 목록, 참여자 체크리스트, 차단 일정이 다시 로드됩니다. 로그아웃하거나 브라우저 탭을 닫으면 관리자 정보가 삭제됩니다.

운영 기본 관리자:

- 한필구 / 01033808374

## 참여자 체크리스트

화면 상단의 `참여자 체크리스트`에서 참여자별 예약 승인 상태, Level, 사용 시간을 확인합니다.

- `예약 승인`: 참여자가 예약 신청을 할 수 있는지 결정합니다.
- `예약 가능`: 승인 상태와 잔여 시간을 기준으로 표시됩니다.
- `사용 시간`: 현재 취소되지 않은 신청 기준으로 계산됩니다.

## 예약 관리

`전체 신청 목록`에서 신청자, 모임명, 공간, 날짜, 상태를 필터링합니다.

자주 쓰는 흐름:

1. `새로고침`을 눌러 최신 신청을 불러옵니다.
2. 신청자나 모임명으로 검색합니다.
3. 취소된 신청을 확인해야 하면 상태 필터를 `취소됨`으로 바꿉니다.
4. 필요한 경우 `취소`를 눌러 관리자 취소를 진행합니다.

## 신청 취소

1. 취소할 신청 행의 `취소` 버튼을 누릅니다.
2. 확인 모달에서 `신청 취소`를 누릅니다.
3. 성공 메시지를 확인합니다.
4. 참여자 화면에서는 기본 목록에서 숨겨지고, `취소된 신청 보기`에서 `취소됨`으로 보입니다.
5. 관리자 화면에서는 `전체 신청 목록`의 상태 필터를 `취소됨`으로 바꾸면 취소된 신청을 확인할 수 있습니다.

## 차단 일정

`관리자 차단 일정` 영역에서 공간, 날짜, 시작시간, 종료시간, 사유를 입력한 뒤 `차단 저장`을 누르면 참가자 예약 화면에 예약 불가 시간으로 반영됩니다.

기존 차단 일정은 `수정`으로 다시 불러와 공간, 날짜, 시간, 사유를 바꿀 수 있습니다. 운영이 끝난 차단 일정은 `해제`를 눌러 비활성 처리합니다. 해제는 삭제가 아니며 운영 이력을 남깁니다.

## 공간 관리

`공간 정보 관리` 영역에서 운영 중 필요한 기본 정보만 수정할 수 있습니다.

자주 쓰는 흐름:

1. 수정할 공간의 `수정 열기`를 누릅니다.
2. 공간명, 상위 공간명, 정원, 대표 이미지 URL, 특징 태그, 설명, 관리자 메모를 필요에 맞게 수정합니다.
3. `수정 저장`을 누릅니다.
4. 참가자 화면 노출을 잠시 막아야 하면 `사용자 숨김`을 누릅니다.
5. 다시 공개하려면 `사용자 노출`을 누릅니다.
6. 운영에서 제외할 공간은 `비활성 처리`를 사용합니다. 비활성 처리 전에는 예약 영향 범위를 한 번 더 확인하세요.

공간 추가와 이미지 목록 관리는 관리자 화면에서 제공하지 않습니다. 운영 데이터 보정이 필요하면 `supabase/manual-sql` 또는 별도 검토된 SQL로 처리합니다.

유지 대상:

- spaces
- space_images
- operating_hours
- admin_blocks

`meetings/sessions` 초기화는 관리자 화면 버튼으로 제공하지 않습니다. 운영 전 초기화가 필요하면 `supabase/manual-sql/03_clear_test_reservations.sql`을 사용합니다.

- [ ] 차단 일정 저장/수정/해제 확인

## 문제 발생 시 확인 순서

1. 참가자가 로그인되지 않으면 `participants`의 이름/전화번호/활성 상태를 확인합니다.
2. 예약 버튼이 비활성화되면 승인 상태, Level 사용량, 운영시간, 기존 예약, 차단 일정을 확인합니다.
3. 관리자 로그인이 되지 않으면 `admins`의 이름/전화번호/활성 상태를 확인합니다.
4. 관리자 목록이 비어 보이면 브라우저 새로고침 후 관리자 화면이 유지되는지 확인하고, `새로고침`을 눌러 `get_admin_participants` RPC 결과를 확인합니다.
5. 신청 취소 후 참여자 화면이 다르면 참여자 화면의 `새로고침`을 누르고, `취소된 신청 보기`를 확인합니다.
6. 공간이 참가자 화면에 보이지 않으면 `공간 정보 관리`의 사용자 노출 상태와 활성 상태를 확인합니다.
