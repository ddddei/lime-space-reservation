# 운영 QA 체크리스트

배포 전 또는 오픈 전 실제 화면에서 확인하는 목록입니다. 당일 간단 점검은 [OPERATION_CHECKLIST.md](./OPERATION_CHECKLIST.md), 문제 발생 시 확인 순서는 [TROUBLESHOOTING.md](./TROUBLESHOOTING.md)를 참고하세요.

## 참가자 화면

- [ ] 등록된 참가자가 이름과 전화번호로 로그인할 수 있음
- [ ] 승인 완료 참가자가 예약 신청을 시작할 수 있음
- [ ] 승인 대기 참가자는 예약 신청이 제한됨
- [ ] 공간 사진이 보임
- [ ] 공간 설명, 운영시간, 특징이 보임
- [ ] 공간 선택 가능
- [ ] 날짜 선택 가능
- [ ] 시간 선택 가능
- [ ] 모임명 입력 가능
- [ ] `모임공간 신청하기`로 신청 완료 가능
- [ ] 신청 후 `내 신청 확인/수정`에 바로 표시됨
- [ ] `새로고침`으로 내 신청 목록 갱신 가능
- [ ] 참가자가 `신청 취소` 가능
- [ ] 취소된 신청은 기본 목록에서 숨겨짐
- [ ] `취소된 신청 보기`에서 취소된 신청 확인 가능
- [ ] Discord 문의 링크가 열림
- [ ] 로그아웃 후 다시 로그인 가능

## 관리자 화면

- [ ] `관리자 모드` 진입 가능
- [ ] 관리자 이름과 전화번호로 로그인 가능
- [ ] 새로고침 후 관리자 화면 유지
- [ ] 로그아웃 가능
- [ ] `참여자 체크리스트` 조회 가능
- [ ] 참가자 예약 승인 상태 확인 가능
- [ ] 참가자 예약 승인 상태 변경 가능
- [ ] `전체 신청 목록` 조회 가능
- [ ] 전체 신청 목록 `새로고침` 가능
- [ ] 관리자 신청 취소 가능
- [ ] 관리자 취소 후 참가자 화면에 취소 상태 반영
- [ ] 취소된 신청은 관리자 상태 필터 `취소됨`에서 조회 가능
- [ ] 차단 일정 저장 가능
- [ ] 차단 일정 수정 가능
- [ ] 차단 일정 해제 가능
- [ ] 차단 일정이 참가자 예약 화면에 반영됨
- [ ] 공간 `사용자 숨김` 가능
- [ ] 공간 `사용자 노출` 가능
- [ ] 공간 `비활성 처리` 가능
- [ ] 공간 설명, 관리자 메모, 정원, 특징 태그 저장 가능
- [ ] 저장 후 새로고침해도 변경사항 유지

## 모바일/화면 확인

- [ ] 모바일 화면에서 가로 스크롤이 생기지 않음
- [ ] 태블릿 화면에서 주요 버튼이 잘림 없이 보임
- [ ] 데스크톱 화면에서 신청 목록과 관리자 영역을 읽기 쉬움
- [ ] 모달 안에서 내용이 잘리지 않음
- [ ] 버튼 문구가 줄바꿈되거나 겹치지 않음
- [ ] 사진 팝업을 닫을 수 있음

## 운영 데이터 확인

- [ ] 참가자 데이터 유지
- [ ] 관리자 데이터 유지
- [ ] 공간 데이터 유지
- [ ] 공간 사진 데이터 유지
- [ ] 운영시간 데이터 유지
- [ ] 차단 일정 데이터 유지
- [ ] 차단 일정 해제는 삭제가 아니라 비활성 처리로 남음
- [ ] 운영 시작 전 필요한 경우 `meetings = 0`
- [ ] 운영 시작 전 필요한 경우 `sessions = 0`

## 배포 전 확인

- [ ] 개발 담당자가 `npm run build` 성공 확인
- [ ] 개발 담당자가 `npm run lint` 성공 확인
- [ ] 운영 환경 변수가 설정되어 있음
- [ ] 실제 운영 URL에서 참가자 로그인 확인
- [ ] 실제 운영 URL에서 관리자 로그인 확인
- [ ] 실제 운영 URL에서 테스트 신청 생성 확인
- [ ] 테스트 신청을 관리자 또는 참가자 화면에서 취소 확인
- [ ] 운영 시작 전 테스트 신청 정리 여부 확인
